# 小游戏平台局域网演示版

这版已经补上：

- 注册 / 登录
- SQLite 数据库（sql.js 落盘到 `data/app.sqlite`）
- 真正用户表
- 密码加密（bcryptjs）
- 用户中心
- 排行榜
- 游戏成绩自动入库
- 管理员后台
- 角色权限（admin / user）
- 端口自动顺延（3000 被占用会自动尝试 3001、3002...）
- **局域网访问**（服务监听 `0.0.0.0`，启动后会直接打印可访问的局域网地址）

## 启动方式

```powershell
npm install
npm start
```

启动后终端会输出实际地址，例如：

```text
服务器已启动：http://localhost:3000
监听地址：http://0.0.0.0:3000
局域网访问地址：
  http://192.168.1.23:3000
```

## 给同学或老师看

1. 你的电脑和对方连同一个 Wi-Fi
2. 终端保持运行，不要关闭
3. 让对方打开终端里打印出的 `http://你的局域网IP:端口`
4. 如果对方打不开，检查 Windows 防火墙是否允许 `node.exe`
